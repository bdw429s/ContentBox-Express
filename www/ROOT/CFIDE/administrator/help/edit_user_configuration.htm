Security > User Manager

 

In the User Manager page, do one of the following:

 

·        Click the user name like in the User column

·        Click the Edit icon in the Action column

 

The User Manager page opens in Edit mode, where you can reconfigure the user account settings such as password, RDS Administrator access, adding sandboxes, and resetting the exposed services for the user.

 

Note: You would not be able to reset the user authentication type in the edit mode.
