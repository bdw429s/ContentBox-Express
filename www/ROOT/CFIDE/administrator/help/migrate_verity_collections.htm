Overview of Verity Collections

Migrate Verity Collections

 

From ColdFusion Administrator, you can migrate Verity collections to Solr by doing the following:

 

1.      Click Data & Services > Migrate Verity Collections.

2.      Select the name of the collection from the Verity collection name to migrate drop-down list.

3.      Specify the name of the new collection in the Name field.

4.      Browse and select the path of the Verity collection that you want to migrate.

5.      Click Submit.

 

Note: After migrating the collection, you need to perform some additional tasks to be able to use the Solr search engine. For more information see Solr Search Support in ColdFusion Developer's Guide.
