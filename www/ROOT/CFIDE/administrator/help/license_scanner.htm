Run the License Scanner

Debugging & Logging > License Scanner

 

The license scanner searches the local subnet to find other running instances of ColdFusion.

Use the license scanner

  1. Click the Run Scanner Now button.
    The ColdFusion Administrator uses universal datagram protocol (UDP) multicast to collect license and version information from all ColdFusion instances running within the subnet.

  2. View the results to determine whether the ColdFusion instances within the subnet are licensed appropriately.
