Overview of Verity Collections

Use Verity K2 Server

Data & Services > Verity K2 Server

 

You can optionally install the Verity server on a different host from ColdFusion. Use the Verity K2 Server page to establish a link to a Verity server that is running on a different host from ColdFusion. If so, use this page to specify the IP address of that host computer.

 

If you purchased the Verity product, you may have to display advanced settings and configure the aliases and ports of the services that ColdFusion uses. However, if you installed the Verity server that ships with ColdFusion on a different host, the IP address alone is usually enough; you do not have to specify advanced settings.

Configure a Verity server that is running on a different host from ColdFusion

  1. In the Verity Host Name box, enter the IP Address of the Verity Server.

  2. (Optional) Click Show Advanced Settings to configure the following information:

    • K2 Admin Alias - The name that K2 assigned to the Verity Admin server.

    • K2 Admin Port - The port used by the Verity Admin server.

    • K2 Server Alias - The name that K2 assigned to the Verity Server server.

    • K2 Server Port - The port used by the Verity Server server.

    • K2 Index Alias - The name that K2 assigned to the Verity Index server.

    • K2 Index Port - The port used by the Verity Index server.

    The Index server is not started if you purchase Verity and run in the default configuration. If this is the case, the cfindex tag does not work, nor can you populate a collection using the ColdFusion Administrator.

  3. Click Submit Changes.
