Server Monitor Settings

Server Monitoring > Server Monitor

 

The Settings section includes the following pages:

 

Aliasing page

The Aliasing page lets you specify aliases for filenames. You can add or delete aliases.

 

Profiling Filter page

The Profiling Filter page lets you specify the file paths to exclude and include in profiling. By default, the Server Monitor collects profiling information about all ColdFusion templates in the webroot directory and its subdirectories and in any directories specified on the Mappings page of the ColdFusion Administrator. However, you may not want to monitor all requests on the server. You specify a path to exclude so that the Server Monitor does not collect profiling information about files in that directory or in any of its subdirectories. This capability is especially useful in restricting monitoring on production servers. Use the Include Paths option to monitor any subdirectories of an excluded directory.

 

Filter Settings page

The Filter Settings page lets you specify the file paths to exclude and include in monitoring and whether to monitor the ColdFusion Administrator. By default, the Server Monitor collects information about all ColdFusion templates in the webroot directory and its subdirectories and in any directories specified on the Mappings page of the ColdFusion Administrator. However, you may not want to monitor all requests on the server. You specify a path to exclude so that the Server Monitor does not collect information about files in that directory or in any of its subdirectories. This capability is especially useful in restricting monitoring on production servers. Use the Include Paths option to monitor any subdirectories of an excluded directory.

 

Refresh Interval page

The Refresh Interval page lets you specify the time interval for refreshing reports and graphs. When the server is running without any problems, you may choose to increase these values. When a problem is reported with the server, or when a developer is profiling an application, a lower value, which results in a faster refresh, may be more useful.
