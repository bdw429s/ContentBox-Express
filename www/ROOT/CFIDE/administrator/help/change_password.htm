Why is Security Important

Change password

Security > Change Password

You can change the password you use to log in to the ColdFusion Administrator.

Change your ColdFusion Administrator password

  1. In the New Password text box, enter a password (up to 50 characters).

  2. In the Confirm Password box, confirm the new password.

  3. Click Submit Changes to save the password.
