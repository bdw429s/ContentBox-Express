Associated Files and Directories tab

Enter information in the Associated Files and Directories in the Archive Wizard

  1. In the Associated Files and Directories tab, specify the full path name of any directory or filename to include in the archive definition.

Tips:

  • Click Browse to quickly locate the full path name of a directory or filename to include in this archive definition.

  • Selecting a directory will recursively retrieve all files and subdirectories.

  • You can type the path followed by *.txt to retrieve all .txt files in that directory; for example: <c:>\*.txt.

  1. To proceed, do one of the following:

  • Click other tabs in the Archive Wizard to continue defining the archive definition.

  • Click Close Window to save this information to the archive definition and exit the Archive Wizard. The name of the archive appears in the Current Archive Definition List.
