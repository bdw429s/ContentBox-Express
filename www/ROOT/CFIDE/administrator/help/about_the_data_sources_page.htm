Data Sources page

Data & Services > Data Sources

 

This page contains two areas:

  • An area for creating a data source and selecting its driver

  • An interactive list of Connected Data Sources

 

For information about creating a data source, see Create a data source.

 

You can edit, verify, and delete a data source from the interactive list of connected data sources, as the following table describes:

 

Column

Description

Actions

Click an icon to edit, verify, or delete a data source.

Data Source Name

Shows the names of the data sources. Click a name to edit that data source.

Driver  

Shows the drivers for corresponding data sources. Click a name to edit the driver's configuration.

Status  

Shows the status of the data source connection as OK or Failed.
