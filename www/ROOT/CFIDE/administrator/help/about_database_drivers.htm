Database drivers overview

JDBC drivers have four types. In general, Type 4 drivers, also known as native-protocol/all-Java drivers, offer superior performance to other JDBC driver types. ColdFusion 9 Enterprise Edition includes a wide variety of Type 4 drivers, and the Standard Edition includes a Type 4 driver for Microsoft SQL Server. ColdFusion also supplies Type 3 drivers for Microsoft Access and for the ODBC Socket, and a Type 2 driver for use with Microsoft Access Unicode databases.
