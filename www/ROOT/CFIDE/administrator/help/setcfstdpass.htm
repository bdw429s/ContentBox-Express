Why Security is Important

Set RDS password

Security > RDS

You can specify a single password to allow access to server resources through RDS. Alternatively, you can limit access by user.

Set the RDS password

 

  1. Select either the RDS authentication type as:

o       Use a single password only option

o       Separate user name and password authentication option to mandate each user to use

o       No authentication needed (not recommended)

  1. In the RDS Single Password section, specify the new password. The minimum character limit is 5 and the maximum character limit is 50.

  2. In the Confirm Password text box, confirm your new password.

  3. Click Submit Changes to save the password.
