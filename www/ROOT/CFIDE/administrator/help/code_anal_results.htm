Code Analyzer results overview

The Code Analyzer report contains two tables:

  • The first table is a summary of Error and Info data.

  • The second table lists the Actions, Feature, and Severity for a Document. You can do the following actions:

  • Sort the table by Feature, Severity, or Document by clicking the appropriate table heading.

  • Display details in a separate window, sorted by feature or by page, by clicking the appropriate button in the Actions column.

  • View the results of a document by clicking the document.
