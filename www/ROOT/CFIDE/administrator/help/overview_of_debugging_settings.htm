Debugging settings overview

The ColdFusion Administrator provides useful debugging information for every application page requested by a browser. You specify debugging preferences using a series of pages:

  • On the Debug Output Settings page, select debugging output options. When you enable debugging, the output appears in a different font after normal page output. For details, see Configure Debug Output Settings.

  • On the Debugging IP Addresses page, restrict access to debugging output. When you enable any of the debugging options, debugging output becomes visible to all users by default, unless you restrict it. For details, see Configure debugging IP addresses.

  • On the Logging Settings page, specify a directory for error log files that can contain useful debugging information. For details, see Configure logging settings.
