Event Gateway Overview

Event gateways settings

Event Gateways > Settings

 

The following table describes the event gateway settings. Click Submit Changes to save your changes. For complete information on event gateways, see the associated chapters in ColdFusion Developer's Guide.

 

Option

Description

Default

Enable ColdFusion Event Gateway Service

Specifies whether the service is enabled. Changing this setting restarts the service.

Enabled

Event Gateway Processing Threads

Specifies the maximum number of threads used to execute ColdFusion functions when an event arrives. A higher number uses more resources, but increases event throughput.

1

Maximum Number of Events to Queue

Specifies the maximum number of events that are allowed on the event queue. If the queue length exceeds this value, gateway events will not be added to the processing queue. This is a global setting for all event gateways.

10

Start/Stop SMS Test Server

Click this button to start and stop the SMS test server.

Stopped
