Server Settings

Configure OpenOffice

Server Settings > Document

 

This page allows you to set the path of your OpenOffice application based on whether it has been installed on a local or remote server.

To configure a local OpenOffice application with ColdFusion:

1.      In the Configure local OpenOffice with ColdFusion section, click Browse Server and select the home directory where OpenOffice is installed.  

2.      Click Add. The OpenOffice directory is configured for ColdFusion.

To configure a remote OpenOffice application instance with ColdFusion:

1        Specify the IP address and port number of the remote host.

2        Click Add. You can also delete the server configuration if configured incorrectly, by clicking Delete.
