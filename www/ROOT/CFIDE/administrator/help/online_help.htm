Online Help

The online Help provided with the ColdFusion Administrator is a 508-compliant HTML Help system. It provides information about using Administrator features, and instructions that guide you through the basic functions of the Administrator.

 

The Help system opens in a web browser window. To open it, click the Help button on a ColdFusion Administrator page.
