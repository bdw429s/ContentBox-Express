Server Monitor Overview

Launching Server Manager AIR Application

Server Manager is an AIR-based application to monitor and manage ColdFusion server settings. From ColdFusion Administrator, you can download the Server Manager AIR application and install it on your local computer.

 

To download the application:

1.      Click Server Monitoring > Server Monitor.

2.      Click Launch Server Manager.

3.      Click Open to run the installer directly. To save the ServerManager.air file on your machine click Save.
