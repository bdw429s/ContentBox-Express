Server Settings

Settings summary

Server Settings > Settings Summary

 

Note: This page appears only for ColdFusion Enterprise Edition.

 

This page lets you view the status of all ColdFusion configuration settings. You can change a setting by clicking its page name, which opens that area of the ColdFusion Administrator.

 

Additionally, you can generate a printable settings summary report in HTML, FlashPaper, or PDF.

 

To save the settings summary as a PDF, click the Save As PDF button on the top-right corner of the page.
