Cluster Manager page

Enterprise Manager > Cluster Manager

 

The Cluster Manager page in ColdFusion Administrator lets you create and manage clusters of JRun servers, each containing the same ColdFusion applications.

 

  1. Add a cluster by typing a name in the Cluster Name box, and clicking Add.
    This creates a cluster and adds it to the Configured Clusters table.

  2. Add servers to the cluster by clicking the Edit icon next to the cluster name in the Configured Clusters table. This displays the Edit Cluster page.

 

Note: The Cluster Manager page appears only if you install the multiserver configuration. It does not display in the server configuration. Nor does it display when running in a J2EE configuration (other than that deployed in the cfusion server of the multiserver configuration).
