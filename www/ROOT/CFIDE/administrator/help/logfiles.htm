Log Files page

Debugging & Logging > Log Files

 

Use this page to perform the following actions:

  • View and search log files

  • Download a log file

  • Archive a log file

  • Delete a log file

 

The following table describes the Log Files page:

 

Area

Description

View Log Files option

Select to view and search multiple log files at the same time.

Actions

Click the icons to view and search the log file content, download the log file to your computer, archive the log file for later use, or delete the log file.

File Name

Lists the log files generated by ColdFusion. To view an individual log file, click its name. Alternatively, you can click the icons to manage the file.

Type

Describes the type of file.

Size

Identifies the size of each log file listed in the table.

Last Modified

Identifies the date that each log file within the table was last modified.

View Log Files button

Click this button to view and search the selected log files.
