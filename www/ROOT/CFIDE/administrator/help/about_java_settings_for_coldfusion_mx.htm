Java settings for ColdFusion

Note: This discussion does not apply when deploying ColdFusion on a J2EE server. When deploying on a J2EE server, use the server-specific method for specifying Java settings.

 

Use the Java and JVM (Java Virtual Machine) Settings page to specify your preferences for how ColdFusion works with Java. For example, you can specify the following details:

  • Location of the Java virtual machine (JVM)

  • Initial and Maximum Heap size to determine the memory range used by the JVM

  • The JVM classpath

  • Any implementation-specific JVM initialization options

  • Location of the ColdFusion cfx.jar file containing interfaces used by Java CFXs
