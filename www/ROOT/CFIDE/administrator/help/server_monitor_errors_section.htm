Server Monitor Errors

Server Monitoring > Server Monitor

 

The Errors section includes the following pages:

 

Requests with Errors page

The Requests with Errors page lists the templates that generated an error. The report includes the path of the template, the number of times errors occurred in that template. For the most recent error, the report indicates the time of the error, the error message, CFML stack traces, and Java stack traces. You can view a list of templates or details about an individual template. The detailed information includes the CFML stack trace.

 

Requests Timed Out page

The Requests Timed Out page lists the templates that timed out. The report includes the path of the template, the number of times the template has timed out, the most recent response time for the template, the time when the template most recently executed, the most recent estimated request size, and the CFML stack trace. A Java stack trace is not provided because timeouts can only occur within CFML. You can view a list of templates or details about an individual template. The detailed information includes the CFML stack trace.
