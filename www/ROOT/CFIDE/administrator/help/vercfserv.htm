Verify that ColdFusion is running

After you install ColdFusion, ensure that the ColdFusion is running by performing the following steps. To run ColdFusion Administrator, you must enable cookies in your browser.

Verify that ColdFusion is running:

  1. Open a web browser.

  2. In the address line, enter the appropriate URL, as follows:

    If ColdFusion is installed on a local server:
    http://localhost/CFIDE/Administrator/index.cfm

    If ColdFusion is installed on a remote server:
    http://    hostname/CFIDE/Administrator/index.cfm

    in which     hostname is the name of the server on which ColdFusion resides. If you are using the internal web server, specify hostname:8500.
    Note:     On UNIX, the ColdFusion Administrator URL is case-sensitive. Ensure that you enter the address as shown.
     

  1. Click Enter.
    If ColdFusion is running, the ColdFusion Administrator opens.
    If you use ColdFusion Administrator security, a password prompt appears.
